# NVIDIA Hub

Пул NVIDIA API-ключей с ротацией прокси, OpenAI-совместимым прокси-сервером и современным веб-интерфейсом (чат + coder IDE).

## Возможности

- **Пул ключей** — добавляй сколько угодно ключей NVIDIA (по одному или массовым импортом), round-robin / least-used стратегии, автоматический cooldown при 401/429, статистика по каждому ключу.
- **Прокси на каждый ключ** — к каждому ключу привязывается несколько прокси (http/socks5), ротация по кругу при каждом запросе, мёртвые прокси отбраковываются автоматически.
- **Открытый порт для общения с ИИ** — сервер поднимает OpenAI-совместимый эндпоинт `http://<host>:8400/v1` (`/v1/chat/completions`, `/v1/models`). Вписывай в codex, Continue, скрипты — все запросы пойдут через пул с ретраями и ротацией.
- **Чат** — стриминг ответов, markdown с подсветкой кода, настройки генерации (temperature, top_p, max tokens), Stop в любой момент.
- **Coder IDE** — файловый менеджер workspace + редактор + агент с function calling (читает/пишет файлы, запускает команды) с живым отображением вызовов инструментов.
- **Вся настройка через интерфейс** — никаких .env и ручных конфигов.

## Запуск

```bash
cd nvidia_hub
./run.sh            # или: venv/bin/python -m app.main
```

Открыть `http://localhost:8400`. Порт меняется переменной `PORT`:

```bash
PORT=9000 ./run.sh
```

## Использование как прокси для внешних тулзов

Base URL: `http://<host>:8400/v1`, API key — любая строка (авторизация не требуется, сервер подставляет ключи из пула).

```bash
curl http://localhost:8400/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{"model": "meta/llama-3.1-8b-instruct", "messages": [{"role":"user","content":"hi"}], "stream": true}'
```

## Структура

- `app/` — бэкенд (FastAPI): `config.py` (хранилище), `pool.py` (пул ключей/прокси), `upstream.py` (запросы к NVIDIA), `agent.py` (coder-агент), `main.py` (эндпоинты)
- `static/` — фронтенд (SPA, ванильный JS)
- `data/config.json` — ключи, прокси, настройки (создаётся автоматически; на проде `chmod 600`)
- `workspace/` — файлы, с которыми работает coder-агент
- `CONTRACT.md` — контракт API

## Требования

Python 3.10+. Зависимости уже в `venv/` (или `pip install -r requirements.txt`).
